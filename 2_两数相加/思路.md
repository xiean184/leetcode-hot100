# 两数相加_简短思路

## 题目核心
给定两个链表 `l1` 和 `l2`，每个链表按**倒序**存储一个非负整数的每一位，要求返回它们相加后的结果链表。

注意：

- 链表是**倒序存储**
- 不是把链表整体转成整数再相加
- 要按“**竖式加法**”逐位处理
- 返回结果也必须是倒序链表

---

## 核心知识点
- **链表遍历**
  - 用 `l1 = l1->next`、`l2 = l2->next` 逐步往后走
- **模拟竖式加法**
  - 每次处理当前一位
- **进位 `carry`**
  - 当前位相加可能产生新的进位
- **虚拟头节点 `dummy`**
  - 方便构造结果链表

---

## 答题技巧
1. 看到“**倒序链表表示数字**”，优先想到 **逐位相加**
2. 看到“**构造新链表**”，优先想到 **dummy 虚拟头节点**
3. 每一轮固定做三件事：
   - 当前位求和
   - 当前节点取 `sum % 10`
   - 进位更新为 `sum / 10`
4. `while` 条件不要只看 `l1` 和 `l2`，还要加上：
   ```cpp
   carry != 0
   ```
   否则最后一位进位会漏掉

---

## 最优思路
同时遍历 `l1` 和 `l2`：

- 如果某条链表当前还有节点，就取它的值
- 如果已经走完，就按 0 处理
- 当前总和：
  ```cpp
  sum = x + y + carry
  ```
- 当前位放：
  ```cpp
  sum % 10
  ```
- 新进位：
  ```cpp
  sum / 10
  ```

然后把这一位接到结果链表后面。

---

## 你做这题时最该注意的点
1. **不能直接把链表转成普通整数**
   - 因为位数可能很长，会溢出
2. **某一条链表先结束，不代表计算结束**
   - 另一条链表可能还有位数
3. **两条链表都结束后，也不一定结束**
   - 如果还有进位，还要再补一个节点
4. **返回的是 `dummy->next`**
   - 不是 `dummy`

---

## 易错点
1. **忘记处理进位**
2. **漏掉最后一个进位节点**
3. **某条链表为空时，没有按 0 处理**
4. **返回 `dummy` 而不是 `dummy->next`**
5. **把题目误解成“大整数相加”而想先转整数**

---

## 标准代码
```cpp
class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        ListNode* dummy = new ListNode(0);
        ListNode* cur = dummy;
        int carry = 0;

        while (l1 != nullptr || l2 != nullptr || carry != 0) {
            int x = 0, y = 0;

            if (l1 != nullptr) {
                x = l1->val;
                l1 = l1->next;
            }

            if (l2 != nullptr) {
                y = l2->val;
                l2 = l2->next;
            }

            int sum = x + y + carry;
            carry = sum / 10;

            cur->next = new ListNode(sum % 10);
            cur = cur->next;
        }

        return dummy->next;
    }
};
```

---

## 一句话记忆
> 这题本质就是用链表模拟竖式加法：当前位等于两数当前节点值加进位，当前节点放 `sum % 10`，进位更新为 `sum / 10`。
