# 合并两个有序链表_简短思路

## 题目核心
给定两个已经升序排列的链表 `list1` 和 `list2`，把它们合并成一个新的升序链表并返回头节点。

注意：

- 不是排序题，而是**利用两个链表本来就有序**
- 返回的是**合并后的链表头节点**
- 链表必须靠 `next` 逐个遍历，不能像数组那样直接用范围 `for`

---

## 核心知识点
- **链表遍历**
  - 链表要用：
    ```cpp
    while (cur != nullptr)
    ```
    来遍历
- **双指针**
  - `list1` 和 `list2` 分别指向两个链表当前节点
- **归并思想**
  - 每次比较两个当前节点，谁小就先接谁
- **虚拟头节点 `dummy`**
  - 方便统一处理头节点，不需要单独讨论“第一个节点接谁”

---

## 答题技巧
1. 看到“**两个有序链表合并**”，优先想到 **归并**
2. 看到“**链表**”，先提醒自己：
   - 不能像 `vector` 一样直接范围 `for`
   - 必须靠 `next` 一步一步走
3. 合并时：
   - 谁小接谁
   - 谁接了谁往后走一步
   - `cur` 也要跟着往后走
4. 其中一个链表走完后，另一个链表剩下部分直接接上

---

## 最优思路
用一个虚拟头节点 `dummy` 和一个尾指针 `cur`。

循环比较 `list1->val` 和 `list2->val`：

- 如果 `list1` 更小，就接 `list1`
- 否则接 `list2`

然后：

- 被接上的那个链表指针往后走
- `cur` 往后走

循环结束后，把还没走完的那条链表直接接到后面。

---

## 你遇到的问题整理

### 问题 1：为什么 `for (ListNode node : list1)` 会报错？
因为 `list1` 的类型是：

```cpp
ListNode*
```

它只是一个链表节点指针，不是：

- `vector`
- 数组
- `string`
- 任何支持范围 `for` 的容器

所以不能写：

```cpp
for (ListNode node : list1)
```

链表必须这样遍历：

```cpp
ListNode* cur = list1;
while (cur != nullptr) {
    ...
    cur = cur->next;
}
```

---

### 问题 2：为什么 `vector<ListNode*> sum;` 不对？
因为你后面往里面放的是：

```cpp
node->val
```

也就是整数值，不是节点指针。

所以如果你真想用暴力法把值取出来再排序，应该定义：

```cpp
vector<int> sum;
```

---

### 问题 3：为什么 `sum.sort();` 不对？
因为 `vector` 没有成员函数：

```cpp
sum.sort();
```

正确写法是标准库算法：

```cpp
sort(sum.begin(), sum.end());
```

---

### 问题 4：为什么 `ListNode* list = nullptr;` 后面直接 `list->val = ...` 不行？
因为 `list` 是空指针，没有指向任何真实节点。

空指针不能这样解引用：

```cpp
list->val = ...
```

如果想建新链表，必须先 `new` 新节点，或者用 `dummy` + `tail` 去接。

---

### 问题 5：为什么最后不能直接返回 `list`？
因为你在构造过程中如果不断：

```cpp
list = list->next;
```

那最后 `list` 已经走到尾巴甚至空了。

题目要求返回的是：

> **整个新链表的头节点**

所以链表题里通常会保留一个：

```cpp
dummy
```

最后返回：

```cpp
dummy->next
```

---

## 易错点
1. **把链表当数组/容器来遍历**
   - 链表不能直接范围 `for`
2. **忘记谁接了谁往后走**
3. **忘记移动 `cur`**
4. **返回 `dummy` 而不是 `dummy->next`**
5. **把“两个有序链表”当成“取值后重新排序”**
   - 这样能做，但不够好，浪费了有序条件

---

## 标准代码
```cpp
class Solution {
public:
    ListNode* mergeTwoLists(ListNode* list1, ListNode* list2) {
        ListNode* dummy = new ListNode(0);
        ListNode* cur = dummy;

        while (list1 != nullptr && list2 != nullptr) {
            if (list1->val <= list2->val) {
                cur->next = list1;
                list1 = list1->next;
            } else {
                cur->next = list2;
                list2 = list2->next;
            }
            cur = cur->next;
        }

        if (list1 != nullptr) {
            cur->next = list1;
        } else {
            cur->next = list2;
        }

        return dummy->next;
    }
};
```

---

## 一句话记忆
> 合并两个有序链表的本质就是归并：谁小接谁，谁接了谁后移，最后返回 `dummy->next`。
